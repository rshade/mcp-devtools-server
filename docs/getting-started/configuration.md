# Configuration

Customize MCP DevTools Server for your project with `.mcp-devtools.json`.

## Configuration File Location

Create `.mcp-devtools.json` in your project root directory.

## Basic Configuration

```json
{
  "lintTools": {
    "enabled": true
  },
  "testTools": {
    "enabled": true
  },
  "goTools": {
    "enabled": true
  }
}
```

For complete configuration options, see the [JSON Schema](.mcp-devtools.schema.json) in the project root.
